# Ensiklopedia Candi Muaro Jambi (KCBN)

Website ensiklopedia interaktif Kompleks Percandian Muaro Jambi dilengkapi AI Pemandu berbasis RAG (Retrieval-Augmented Generation).

## Fitur
- Ensiklopedia 9 candi utama KCBN
- AI Pemandu Wisata berbasis RAG + Mistral
- Repositori artikel penelitian ilmiah
- Berita & informasi terkini
- Informasi kunjungan (tiket, jam buka, guide)

## Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js + Express
- AI: Ollama (Mistral + nomic-embed-text)
- RAG: Cosine Similarity Search

## Cara Menjalankan
1. Install dependencies: `npm install`
2. Jalankan Ollama: `ollama serve`
3. Jalankan server: `npm start`
4. Buka browser: `http://localhost:5000`
